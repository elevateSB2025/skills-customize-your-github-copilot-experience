
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a Hangman-style word game to practice Python control flow, string handling, and game logic.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a word-guessing game that lets a player guess letters until they either uncover the hidden word or run out of attempts.

#### Requirements
Completed program should:

- Choose a random word from a predefined list.
- Ask the player to guess one letter at a time.
- Show the current word progress with underscores for unknown letters, for example: `_ a _ g _ a _`.
- Track incorrect guesses and remaining attempts.
- End the game with a win message when the word is fully guessed.
- End the game with a lose message when attempts reach zero.

### 🛠️ Add Game Feedback

#### Description
Enhance the game so the player receives clear feedback after each guess.

#### Requirements
Completed program should:

- Display the letters that have already been guessed.
- Inform the player whether their guess was correct or incorrect.
- Show the number of remaining guesses.
- Prevent duplicate guesses from counting against the player.

### 🛠️ Optional Challenge: Word Guessing or Difficulty

#### Description
Add an optional feature to make the game more engaging.

#### Requirements
Choose one or both options:

- Allow the player to guess the full word at any time.
- Add difficulty levels with different word lists or attempt counts.
- Provide a final summary of the game outcome and the correct word.
